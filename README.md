# CARTEIRA DE FINANÇAS
 Projeto de uma carteira de financas pessoal, criado com HTML e CSS. A ideia e organizar informacoes como saldo, compras, gastos fixos e poupanca em uma tela simples, responsiva e facil de usar.
 
# Objetivo
O objetivo do projeto e praticar desenvolvimento web criando uma interface de controle financeiro. No futuro, o projeto tera funcionalidades com JavaScript para cadastrar valores, calcular gastos e atualizar os cards automaticamente.

# Funcionalidades Planejadas
- Visualizar saldo atual
- Registrar compras
- Controlar gastos fixos
- Criar uma meta do mês
- Atualizar valores dinamicamente com JavaScript
- Adaptar a tela para computador, notebook, tablet e smartphone
